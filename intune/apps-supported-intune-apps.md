# <a name="microsoft-intune-protected-apps"></a>Aplikace s ochranou Microsoft Intune  

Spravované aplikace jsou povolené s bohatou sadou zásad ochrany mobilních aplikací Intune. Spravované aplikace vám umožní:
- Omezit kopírování a vkládání a uložení – jako funkce
- Konfigurace webových odkazů pro otevírání v zabezpečené prohlížeče Microsoft
- Povolení použití více identit a podmíněného přístupu na úrovni aplikace
- Použít zásady ochrany před únikem informací bez správy uživatelských zařízení
- Povolit ochranu aplikací bez nutnosti registrace
- Povolit ochranu aplikací pro zařízení spravovaná pomocí nástroje EMM 3. stran

> [!NOTE]
> Pro vaše aplikace – obchodní klienta, můžete začlenit pomocí správy mobilních aplikací [sadu Intune App Software Development Kit](app-sdk.md) (SDK), nebo [App Wrapping Tool for iOS](app-wrapper-prepare-ios.md) a [aplikace Nástroj pro zabalení pro Android](app-wrapper-prepare-android.md). 

Následující tabulka obsahuje podrobnosti o podporovaných partnera a aplikace od Microsoftu, které se běžně používají v Microsoft Intune.
