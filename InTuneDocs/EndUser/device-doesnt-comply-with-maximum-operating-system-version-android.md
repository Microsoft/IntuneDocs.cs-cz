# <a name="android-device-doesnt-comply-with-the-maximum-operating-system-version"></a>Zařízení s Androidem nevyhovuje maximální verzi operačního systému.

Zařízení Android nevyhovuje zásadám vašeho správce IT, které se týkají maximální verze operačního systému. Požádejte o pomoc správce IT. Kontaktní údaje najdete na [webu Portál společnosti](http://portal.manage.microsoft.com).
